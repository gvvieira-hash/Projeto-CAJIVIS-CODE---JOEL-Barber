# 💈 Joel Barber — Sistema de Agendamento para Barbearia

## 📌 Sobre o Projeto

O **Joel Barber** é um sistema web moderno desenvolvido para facilitar o agendamento de horários em uma barbearia.
O projeto foi criado com foco em experiência do usuário, design profissional e navegação intuitiva.

A plataforma permite que clientes escolham serviços, barbeiros, datas e horários disponíveis de forma rápida e prática.

---

# 🚀 Funcionalidades

✅ Agendamento de serviços
✅ Seleção de múltiplos serviços
✅ Escolha do barbeiro
✅ Escolha de data e horário
✅ Página de confirmação dinâmica
✅ Sistema de fila de espera
✅ Avaliação da barbearia
✅ Interface responsiva
✅ Navegação entre páginas
✅ Salvamento das escolhas com LocalStorage
